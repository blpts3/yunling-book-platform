# 交大引路人 - SWJTU Mentor Connect

西南交通大学学长学姐互助平台 - 打破信息差,连接每一位交大学子与优秀导师

## 项目简介

这是一个为西南交通大学学生打造的互助平台,主要功能包括:

- **导师库**: 浏览优秀学长学姐导师,查看他们的背景和成就
- **信息中心**: 获取保研、留学、考研、就业的一手经验和资讯
- **智能匹配**: 根据你的专业和目标,智能推荐最适合的导师
- **社团大全**: 了解学校各类社团信息

## 技术栈

- React 18 + TypeScript
- Vite 6 (构建工具)
- Tailwind CSS (样式框架)
- React Router (路由管理)
- Lucide React (图标库)

## 开发指南

### 前置要求

- Node.js 18+ 
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173 查看应用

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 功能特性

### 1. 首页
- Hero区域展示平台价值主张
- 统计数据展示
- 精选导师推荐
- 成功案例展示
- CTA行动号召

### 2. 导师库
- 搜索功能(姓名、专业、院校)
- 标签筛选
- 导师卡片展示(头像、背景、成就)
- 在线状态标识
- 联系导师功能

### 3. 信息中心
- 经验分享文章(保研/留学/考研/就业)
- 社团信息展示
- 分类浏览和搜索
- 文章详情查看

### 4. 智能匹配
- 多步骤表单收集用户需求
- 基于专业和兴趣的智能匹配
- 匹配结果展示
- 一键联系导师

## 设计系统

采用蓝色系主色调,代表学术、信任和知识:

- **主色**: 深蓝色 (#1e40af) - 学术与信任
- **辅助色**: 天蓝色 (#3b82f6) - 活力与希望
- **强调色**: 橙色 (#f97316) - 温暖与行动
- **成功色**: 绿色 - 完成与确认
