# 🚀 云凌书驿 - 快速开始指南

## 第一步：创建Supabase项目（5分钟）

1. **访问Supabase**
   - 打开 https://supabase.com
   - 点击 "Start your project"

2. **注册/登录**
   - 可以使用GitHub账号快速登录

3. **创建新项目**
   - 点击 "New project"
   - 填写：
     - **Name**: `yunling-book`
     - **Database Password**: 设置一个密码（请保存！）
     - **Region**: 选择 `Singapore`（离中国最近）
   - 点击 "Create new project"
   - 等待2-3分钟创建完成

4. **获取API密钥**
   - 进入项目后，点击左侧 "Settings" → "API"
   - 复制以下两个值：
     ```
     Project URL: https://xxxxxxxxxxxxx.supabase.co
     anon public: eyJhbGc...（很长的字符串）
     ```

## 第二步：执行数据库Schema（2分钟）

1. **打开SQL编辑器**
   - 在Supabase左侧菜单点击 "SQL Editor"

2. **执行Schema**
   - 点击 "New query"
   - 打开项目文件夹中的 `supabase-schema.sql` 文件
   - 全选复制所有内容
   - 粘贴到SQL编辑器
   - 点击右下角 "Run" 按钮

3. **确认成功**
   - 看到 "Success. No rows returned" 表示成功

## 第三步：配置本地环境（1分钟）

1. **创建.env文件**
   ```bash
   # 在项目根目录执行
   copy .env.example .env
   ```

2. **编辑.env文件**
   用记事本打开 `.env`，填入：
   ```env
   VITE_SUPABASE_URL=https://你的项目ID.supabase.co
   VITE_SUPABASE_ANON_KEY=你的anon密钥
   VITE_ADMIN_EMAIL=你的邮箱@gmail.com
   ```

## 第四步：测试运行（1分钟）

```bash
npm run dev
```

访问 http://localhost:5173/auth 测试注册登录功能

## 第五步：部署到Netlify（10分钟）

### 方法A：最简单（推荐新手）

1. **安装Netlify CLI**
   ```bash
   npm install -g netlify-cli
   ```

2. **构建项目**
   ```bash
   npm run build
   ```

3. **登录Netlify**
   ```bash
   netlify login
   ```
   会自动打开浏览器，授权登录

4. **部署**
   ```bash
   netlify deploy --prod --dir=dist
   ```
   - Site name: 输入一个名字如 `yunling-book-2024`
   - 等待部署完成

5. **添加环境变量**
   - 在Netlify控制台进入你的网站
   - Site settings → Environment variables
   - 添加：
     - `VITE_SUPABASE_URL`
     - `VITE_SUPABASE_ANON_KEY`
   - 重新部署一次（Deploy site）

### 方法B：通过GitHub（推荐长期使用）

1. **推送到GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   # 在GitHub创建仓库后
   git remote add origin https://github.com/你的用户名/yunling-book.git
   git push -u origin main
   ```

2. **在Netlify连接GitHub**
   - 访问 https://netlify.com
   - "Add new site" → "Import an existing project"
   - 选择GitHub，授权
   - 选择你的仓库
   - Build command: `npm run build`
   - Publish directory: `dist`
   - 添加环境变量
   - 点击 "Deploy site"

## 第六步：设置管理员账户

1. **注册第一个账户**
   - 使用您在 `.env` 中配置的邮箱注册

2. **设置为管理员**
   - 在Supabase控制台 → SQL Editor
   - 执行：
     ```sql
     UPDATE users SET role = 'admin' WHERE email = '你的邮箱';
     ```

3. **测试管理后台**
   - 登录后访问 `/admin`

## 🆘 常见问题

**Q: 注册后无法登录？**
A: Supabase默认需要邮箱验证。在Supabase控制台关闭：
- Authentication → Settings → Email Auth
- 关闭 "Enable email confirmations"

**Q: 部署后看不到数据？**
A: 需要在Netlify环境变量中添加Supabase配置，并重新部署。

**Q: 如何绑定自己的域名？**
A: 在Netlify控制台 → Domain settings → Add custom domain
